# jjeff-typescript-template

## Description

This is a template for a typescript project. It includes the following:

- [x] Typescript
- [x] Mocha
- [x] Chai
- [x] ESLint
- [x] Prettier
- [x] Semantic Release
- [x] JSDoc
- [x] Github Actions
- [x] VS Code Formatting, Linting, and Validation Settings
